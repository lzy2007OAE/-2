# 速跑RunFast APP响应式展示页面 课程作业
## 一、页面包含5个要求模块（对应评分点）
1. APP介绍区域：页面顶部hero板块，包含APP名称+宣传标语
2. 核心功能展示区：4个跑步功能卡片，带悬浮上浮交互动画
3. APP技术规格表格：展示系统、安装包、定位等参数
4. 用户评价板块：3条真实用户体验评价，移动端自动竖排布局
5. 会员订阅区域：月/年两种会员定价，点击购买按钮弹出交互弹窗

## 二、技术说明
1. 技术栈：HTML5 + CSS3（Grid/Flex响应式）+ 原生JavaScript
2. 响应式：电脑/平板/手机自动切换布局
3. 交互效果：按钮点击弹窗、卡片悬浮、滚动淡入动画
4. 托管方式：GitHub Pages 静态网页在线预览

## 三、预览方式
开启GitHub Pages后访问生成链接即可查看完整页面
